# WordCloud App

## Installation

To install the project, clone the repository on your machine then run :
```
npm install
```

## Running the application
To run :
```
npm start
```

Then open a browser at this url :
```
http://localhost:8000
```

To open the application in debug mode :
```
http://localhost:8000/debug.html
```

To run tests :
```
npm test
```
